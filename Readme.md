## 使用方法：

**随机卡组生成器需配合YGOPro22使用**

在任意路径下运行随机器可随机器

填写YGoPro2路径：X:\XXXX\YGOPro2

使用“验证血统”按钮可以进行卡组系列更换

使用“Yu-Gi-Oh!”按钮将随机卡组存放至YGOPro2目录
